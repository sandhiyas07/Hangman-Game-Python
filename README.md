# 🎯 Hangman Game

## 📖 Description

A simple and interactive Hangman Game built using Python. Players try to guess the hidden word one letter at a time before running out of attempts.

## ✨ Features

* 🎲 Random word selection
* 🔤 Letter-by-letter guessing
* ❤️ Limited attempts
* 🏆 Win and Lose conditions
* 💻 Console-based gameplay

## 🛠️ Technologies Used

* Python

## 🚀 How to Run

1. Open the project in VS Code.
2. Run the Python file.
3. Start guessing letters to find the hidden word.
